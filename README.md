## Installation
```
$ git clone git@github.com:mayronceccon/olist-python-labs-project.git
$ cd olist-python-labs-project
$ virtualenv .venv
$ source source .venv/bin/activate
$ pip install -r requirements.txt
```
---
### Usage
```
$ python main.py
```
